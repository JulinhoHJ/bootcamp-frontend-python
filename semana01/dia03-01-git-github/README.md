# Bootcamp Frontend G30

Este repositorio nos sirve para aprender el flujo de trabajo de GIT y Github.


### Comandos de configuración:

* git config --global user.name "Julinho HJ"
* git config --global user.email "wjulinho.hj@gmail.com"

### Comandos básicos:

* git init

* git status : Para ver el estado de nuestro repositorio

* git add .
* git add -A
* git add index.html

* git commit -m "subir archivos de la primera semana"

* git log : lista los commit que hemos confirmado
* git log --oneline
* git log --oneline --graph

* git show
* git remote add origin https://...
* git remote remove origin

* git branch -M main

* git push -u origin main
* git push origin main
* git push

* git pull origin main
* git pull

* git checkout ejemplo
* git checkout -b ejemplos
